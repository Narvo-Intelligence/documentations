## 1. Brand Architecture

### 1.1 The Three Brands

| Brand | Type | Audience | Personality |
|-------|------|----------|-------------|
| **Narvo Intelligence** | Parent / Corporate | Investors, legal, partners | Modern-institutional, forward-thinking, precise |
| **Narvo** | Consumer App (B2C) | End users, listeners, diaspora | Bold, vibrant, culturally-alive |
| **Narvo Platform** | Business (B2B) | Developers, publishers, enterprise | Technical-modern, confident, distinctive |

### 1.2 The Unifying Principle: One Font Family, Three Palettes

All three brands share **Clash Display** (display), **Satoshi** (body), and **Geist Mono** (code). Brands differentiate through colour and context, not font families. This follows the Apple/Google/Stripe approach.

### 1.3 When to Use Each Brand

| Context | Brand |
|---------|-------|
| App Store, in-app UI, consumer marketing, social media | **Narvo** |
| API docs, developer portal, B2B website, sales materials | **Narvo Platform** |
| Investor deck, press releases, contracts, corporate website | **Narvo Intelligence** |
| Internal docs mentioning both products | **Narvo Intelligence** |

## 3. Narvo — Consumer Brand

> *"The news should feel like it was made just for you."*

**Tagline:** "The Local Pulse, Refined."

### 3.1 Colour Palette

| Role | Hex | Usage |
|------|-----|-------|
| Background | `#FFFCF5` | App background |
| Surface | `#FFFFFF` | Cards, modals |
| Surface Raised | `#FFF3E0` | Hover, highlights |
| Primary | `#D4520A` | CTAs, brand anchor |
| Primary Muted | `#FFF1E8` | Tag backgrounds |
| Accent | `#0A6847` | Verification, secondary |
| Text Primary | `#1A1207` | Headlines, body |
| Text Secondary | `#6B5A40` | Labels, metadata |

Deep burnt orange — Lagos sunsets, market energy, confidence. Forest green accent grounds it.

### 3.2 Typography

| Role | Font | Usage |
|------|------|-------|
| Display | **Clash Display** 500–700 | Headers (28px+), story titles (16px+) |
| Body | **Satoshi** 400–700 | Body, labels, buttons |
| Mono | **Geist Mono** 400 | Timestamps, truth tags |

Typography is deliberately **LARGE**: 28px+ headers, 52px+ play buttons.

### 3.3 Design Principles

- **Bold and Large:** 48px+ touch targets. Magnetic, not polite.
- **Rounded:** Card radius 14–20px. Button radius 10–14px.
- **Warm Elevation:** Orange-tinted shadows.
- **Audio-Forward:** 52px+ play. Musical waveform. Prominent player.
- **Light Mode Default.** Dark mode available.
- **African Energy:** Photography reflects African culture.

### 3.4 Tone: "Welcome, Oga" not "Welcome, User." "Oya, Play" not "Play Audio."

### 4.3 Design Principles

- **Dark Mode Primary.** Desktop-first.
- **Data-Forward:** Metrics, code samples in hero.
- **Minimal Radius:** 6–10px. Subtle.
- **Code as Content.** Copy-paste-ready.
- **Trust Signals:** Logos, uptime, SLA.

## 5. Cross-Brand Rules

### Shared: Clash Display + Satoshi + Geist Mono across all brands.

### Never Do

- Never mix consumer/platform palettes
- Never show "Narvo Intelligence" in consumer contexts
- Never use consumer tone in Platform/Intelligence materials
- Never substitute a different display font

### Logo Direction

| Brand | Style |
|-------|-------|
| Narvo Intelligence | Clash Display SemiBold. Indigo→mint gradient mark. |
| Narvo | Clash Display Bold. Lowercase "narvo." |
| Narvo Platform | "narvo" semibold + "platform" regular. |

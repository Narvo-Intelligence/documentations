# Legacy Documentation (Archived)

> These documents are from the pre-pivot era (Swiss Grid design system, Broadcast Entity concept). They are archived for reference. The [current/](../current/) directory contains the active documentation.

## What changed

The March 2026 pivot moved Narvo from a "technical instrument" metaphor (Swiss Grid, 0px radius, 1px borders, Space Grotesk + Inter) to a "personal companion" experience (bold, warm, Clash Display + Satoshi, burnt orange palette, rounded cards, large UI).

The company also restructured from "Narvo" (single product) to "Narvo Intelligence" (parent) with two product lines: Narvo (B2C consumer app) and Narvo Platform (B2B NaaS).
